Publishing & updating your website

Your site is a folder of ready-to-go files. This guide shows the two simplest ways to put it online at appasanimd.com — and how to make changes afterward. No coding required.

What you have Start here

A downloaded ZIP named _site. Double-click to unzip it first — everything lives inside that one folder (all pages, photos, and the assets folder). Keep them together; that's the whole website.

Then choose a path below. Option A is fastest to go live today. Option B is best for easy ongoing edits.

Option A  ·  Netlify Drop ~2 minutes · no account linking

The quickest way to see your site live on your domain.

  1. Unzip the downloaded folder (double-click it).
  2. In your browser, go to app.netlify.com/drop and sign in (free).
  3. Drag the unzipped folder onto the drop area. Your site goes live instantly on a temporary web address.
  4. Open Site configuration → Domain management → Add a domain, enter appasanimd.com, and follow the on-screen DNS steps with your domain registrar.
↻
To update later: ask me for a fresh copy, then drag the new folder onto the same Netlify site (Deploys → drag-and-drop). It replaces the old version in seconds.

Option B  ·  GitHub + Netlify Best for ongoing edits · auto-publishes

A one-time setup that makes every future change publish automatically. You create the repository once (my access to GitHub is read-only), then connect Netlify to it.

Part 1 — Put the site on GitHub (one time)

  1. Go to github.com → click the + (top-right) → New repository.
  2. Name it appasani-md, leave the defaults, and click Create repository.
  3. On the next screen, click the link "uploading an existing file."
  4. Unzip your folder, then drag all the files inside it (including the assets folder) into the browser window.
  5. Click Commit changes. Your website now lives on GitHub.

Part 2 — Connect Netlify (one time)

  1. Go to netlify.com and sign in (free).
  2. Click Add new site → Import an existing project → Deploy with GitHub.
  3. Authorize Netlify, then pick your appasani-md repository.
  4. Leave the build settings empty and click Deploy. Your site is live.
  5. Add appasanimd.com under Domain management, as in Option A.
↻
To update later: edit any file right on GitHub (open the file → pencil icon → make changes → Commit), or have me send you updated files to drag in. Netlify republishes automatically within a minute — nothing else to do.

Two things to do once you're live

Activate the contact form

From the live site, submit one test inquiry. The first time, you'll get a one-time confirmation email — click it to turn the form on. After that, every inquiry emails to info@appasanimd.com and is logged in your Formspree dashboard.

Your domain is pre-set

The site's search-engine tags already point to appasanimd.com, so there's nothing to edit when you connect the domain. Just add it in Netlify and update DNS.
